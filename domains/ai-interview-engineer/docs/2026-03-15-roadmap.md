# AI 算法岗 / AI 开发工程师面试学习路线

## 你需要具备哪些知识

先把知识分成 6 大块：

1. 编程与算法
   - Python 基础、面向对象、常用库
   - 数据结构与算法
   - LeetCode/手撕代码

2. 数学与机器学习基础
   - 线性代数、概率统计、微积分
   - 监督学习、无监督学习、损失函数、正则化
   - 模型评估、过拟合、偏差方差、交叉验证

3. 深度学习与主流模型
   - MLP、反向传播、激活函数、归一化
   - 优化器、学习率、BatchNorm、Dropout、残差结构
   - CNN、RNN/LSTM、Attention、Transformer

4. AI 方向知识
   - 如果偏算法岗：CV / NLP / 推荐 / 多模态，至少精通一个
   - 如果偏 AI 开发岗：LLM、RAG、Agent、工具调用、Prompt、向量库、检索链路

5. 工程与系统能力
   - PyTorch 训练与调试
   - 数据处理、实验管理、模型部署
   - 后端接口、并发、缓存、数据库、系统设计
   - 推理优化、MLOps、监控

6. 项目与面试表达
   - 讲清楚做过的项目
   - 能解释模型选择、指标、误差分析、线上问题
   - 能说清楚技术 trade-off

## 建议学习顺序

### 第一阶段：补地基

- Python
- 数据结构与算法
- 数学基础
- 机器学习基础

### 第二阶段：补深度学习主干

- 神经网络训练流程
- 常见网络结构
- Transformer
- PyTorch 训练实践

### 第三阶段：岗位分叉

- 算法岗：CV / NLP / 推荐 选一个主方向深挖
- AI 开发岗：LLM、RAG、Agent、后端与部署深挖

### 第四阶段：面试冲刺

- 项目复盘
- 高频八股
- 手撕代码
- 模拟面试

## 面试视角下的优先级

如果你基础弱，最先补：

1. Python
2. 数据结构与算法
3. 机器学习基础
4. 深度学习基础
5. Transformer 和 LLM
6. 一个能讲透的项目

如果你已经会训练模型，最先补：

1. 项目表达
2. 系统设计与部署
3. RAG / Agent / 推理优化
4. 手撕代码

## 第一轮学习主题建议

第一轮先学这 4 个概念：

1. 监督学习、损失函数、梯度下降到底在做什么
2. 过拟合、正则化、偏差方差怎么区分
3. 反向传播为什么能训练神经网络
4. Transformer 为什么比 RNN 更适合大模型
